# منصة الرصد الإعلامي لقطاع الإعلام

الجهة المالكة: قطاع التطوير التقني  
الحقوق: جميع الحقوق محفوظة لقطاع التطوير التقني

تطبيق جديد مستقل بالكامل مبني من الصفر باستخدام `Streamlit` مع قاعدة بيانات محلية `SQLite` ومحرك رصد متعدد المصادر وتحليل وتصنيف ودمج قضايا وتقارير احترافية ومساعد داخلي بعنوان `اسأل الذكاء الاصطناعي`.

## المزايا

- بحث متعدد المصادر: ويب عام، أخبار، مواقع رسمية، `X`، `YouTube`، وروابط مباشرة.
- دعم `Google Dorking` عبر حقل استعلام متقدم.
- دعم الصور والفيديو عند الإمكان مع OCR واستخراج نص فيديو YouTube.
- تصنيف المحتوى إلى: شكوى، استغاثة، طلب مساعدة، انتقاد، إشادة، خبر محايد، غير ذي صلة، مكرر.
- حساب درجة خطورة لكل نتيجة وقضية.
- دمج النتائج المتشابهة في قضايا `Cases`.
- تقارير فعلية بصيغ `PDF` و`DOCX` و`CSV` و`JSON`.
- AI Assistant داخلي لا يجيب إلا من البيانات المخزنة داخل النظام مع ذكر الأدلة والثقة.

## هيكل المشروع

- `app.py`: نقطة التشغيل.
- `monitoring_app/storage.py`: قاعدة البيانات المحلية.
- `monitoring_app/services/`: البحث، التحليل، دمج القضايا، التقارير، المساعد.
- `monitoring_app/ui/`: الواجهة والثيم.
- `data/`: قاعدة البيانات المحلية عند التشغيل.
- `exports/`: التقارير المولدة.

## التشغيل

1. `pip install -r requirements.txt`
2. `streamlit run app.py`

## بيانات الدخول الافتراضية

- اسم المستخدم: `admin`
- كلمة المرور: `Admin@123`

## ملاحظات تشغيلية

- يوصى باستخدام Python 3.10 أو أحدث.
- دعم `X` يعتمد على النتائج العامة المفهرسة على الويب، لأن الوصول المباشر لمحتوى X قد يتأثر بقيود المنصة.
- OCR واستخراج نص الفيديو يعملان عند توفر الوسائط العامة أو النصوص المصاحبة.
- في بيئات الاستضافة السحابية قد يتم تعطيل OCR تلقائيًا إذا لم تتوفر مكتبته الاختيارية، لكن التطبيق سيستمر في العمل دون كسر التشغيل.
- عند النشر على `Streamlit Community Cloud` اختر إصدار Python من `Advanced settings` داخل منصة النشر إذا ظهرت مشكلة توافق في الاعتمادات.
